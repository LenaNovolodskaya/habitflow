ЭКЗАМЕНАЦИОННЫЙ ПРОЕКТ
1. НАЗВАНИЕ ПРОЕКТА
HabitFlow — трекер полезных привычек

2. ОСНОВНЫЕ КОМПОНЕНТЫ (3 МИКРОСЕРВИСА)

2.1. Сервис пользователей (User Service)
- Регистрация и вход пользователей
- Создание, редактирование и удаление привычек
- Управление профилем пользователя

Примеры привычек: "пить 2 литра воды", "утренняя зарядка", "читать 30 минут"

2.2. Сервис отслеживания (Tracking Service)
- Отметка выполнения привычек за день
- Подсчет дней подряд (streak)
- Хранение истории выполнения
- Проверка уникальности отметок (нельзя отметить дважды в день)

2.3. Сервис статистики (Stats Service)
- Расчет процента выполнения за неделю/месяц
- Отображение текущих серий выполнения
- Общая статистика по всем привычкам
- Формирование отчетов о прогрессе

3. ВЗАИМОДЕЙСТВИЕ КОМПОНЕНТОВ

Пример: "Пользователь отмечает выполнение привычки"
- Пользователь заходит в веб-приложение (Vue.js)
- Нажимает кнопку "Отметить" у привычки "Пить воду"
- Фронтенд (Vue.js) отправляет запрос на Tracking Service:

javascript
POST http://localhost:8002/api/completions
{
  "userId": 1,
  "habitId": 5,
  "date": "2024-01-15"
}
Tracking Service (Node.js) получает запрос и:
- Проверяет существование пользователя (запрос к User Service)
- Проверяет существование привычки (запрос к User Service)
- Записывает отметку в PostgreSQL
- Считает: "Эта привычка выполняется 5 дней подряд"
- Tracking Service сообщает Stats Service об обновлении
- Stats Service (Node.js) обновляет статистику пользователя
- Фронтенд (Vue.js) получает ответ и обновляет интерфейс

4. ТЕХНОЛОГИИ РАЗРАБОТКИ

Backend (серверная часть)
- Язык: JavaScript (Node.js)
- Фреймворк: Express.js
- База данных: PostgreSQL
- Работа с БД: библиотека pg (node-postgres)
- Аутентификация: JWT-токены
- Валидация: библиотека joi
- Логирование: console.log или библиотека winston

Frontend (клиентская часть)
- Язык: JavaScript
- Фреймворк: Vue.js 3
- Управление состоянием: Vuex или Composition API
- HTTP запросы: библиотека axios
- Стили: CSS 
- Маршрутизация: Vue Router
- Иконки: Font Awesome или Material Icons

Инфраструктура
- Контейнеризация: Docker
